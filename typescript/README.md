# TypeScript Experiments

This directory contains test files and experiments for learning TypeScript. 
Personal TypeScript utilities and web apps for Raspberry Pi 5

## Overview

A playground for experimenting with TypeScript concepts, syntax, and tools. Ideal for practicing TypeScript basics and advanced features.

## Getting Started

### Prerequisites

- Node.js (recommended version 14 or higher)
- TypeScript (`npm install -g typescript`)

### Installation

Navigate to this directory from the playground repository root: 
```bash
cd typescript
# Install dependencies if any are added in the future

## Usage

You can run TypeScript files after compiling them:
```bash
tsc hello.ts
node hello.js
```

Or use `ts-node` for direct execution (install with `npm install -g ts-node`):
```bash
ts-node hello.ts
```

## File Structure

- `hello.ts` – Example TypeScript file for experiments
- `tsconfig.json` – TypeScript compiler configuration
- `game-demo/` – Directory for game-related TypeScript experiments
